## Quick start

### Backend
```
cd backend
npm install
npm start
```
Server: http://localhost:4000

### Frontend
```
cd frontend
npm install
npm start
```
App: http://localhost:3000

- frontend/: React frontend
- report.md: Project report (convert to PDF as required)
- UML/: Mermaid diagrams
